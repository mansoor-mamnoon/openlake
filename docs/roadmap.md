# Roadmap

v0 is single-node and focused on shipping a delightful local experience.

Near-term milestones:

* Postgres metadata schema + migrations
* Delta-like transaction log + snapshots
* CSV ingest to Parquet + versioned commits
* DuckDB SQL query endpoint + UI query page
* Streaming ingestion micro-batches (Redpanda)
* YAML pipelines + DAG runner
* Experiment tracking (MLflow-lite)
* Dataset version linkage + run reproducibility
